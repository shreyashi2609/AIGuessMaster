# AI Guess Master

AI Guess Master is an interactive Number Guessing Game powered by a smart AI assistant. Built with a Flask backend and a modern HTML/CSS/JS frontend, it features real-time feedback, intelligent hints, and a polished user interface.

## Key Highlights
- **Smart AI Hints**: Get strategic advice and range suggestions to solve the puzzle faster.
- **Responsive UI**: A beautiful, mobile-friendly interface with smooth animations.
- **Full Stack Architecture**: clean separation of concerns using RESTful APIs.

Perfect for users who enjoy logic puzzles and developers looking for a clean example of a Flask + Vanilla JS application.
